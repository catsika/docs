description: "Official Node.js SDK for Keymint license management with TypeScript support and comprehensive examples."
---

The official Keymint Node.js SDK provides a simple interface for integrating license management into your Node.js applications.

## Installation

Install the SDK using npm:

```bash
npm install keymint-nodejs
```

## Quick Start

```javascript
const Keymint = require('keymint-nodejs');

const keymint = new Keymint({
  accessToken: 'YOUR_ACCESS_TOKEN'
});

// Create a customer
const customer = await keymint.customer.create({
  name: 'John Doe',
  email: 'john@example.com'
});

// Create a license key
const license = await keymint.key.create({
  productId: 'your-product-id',
  maxActivations: '5',
  newCustomer: {
    name: 'License Customer',
    email: 'customer@example.com'
  }
});

// Activate a license
const activation = await keymint.key.activate({
  productId: 'your-product-id',
  licenseKey: 'LICENSE-KEY-HERE',
  hostId: 'unique-machine-id',
  deviceTag: 'user-device-name'
});
```
