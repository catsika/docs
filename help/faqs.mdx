## Licensing

<AccordionGroup>
  <Accordion title="What is software licensing?">
    Software licensing is a legal mechanism that governs the use and distribution of software. It defines the terms under which users can install, use, and sometimes modify software. Keymint provides the technical tools to implement and enforce these license agreements in your software products.
  </Accordion>
  <Accordion title="What licensing models does Keymint support?">
    Keymint supports four key licensing models: node-locked licenses, perpetual licenses, subscription-based licenses, and trial licenses. Each model offers flexibility for different software distribution strategies.
  </Accordion>
</AccordionGroup>

## Technical

<AccordionGroup>
  <Accordion title="What programming languages does Keymint support?">
    Keymint provides a REST API that can be used with any programming language capable of making HTTP requests. We offer comprehensive API documentation and examples for Python, JavaScript, Java, C#, and more.
  </Accordion>
  <Accordion title="How secure is the Keymint API?">
    Our API uses industry-standard security protocols including HTTPS, JWT authentication, and rate limiting. We implement robust encryption and provide detailed access logs to ensure maximum security.
  </Accordion>
</AccordionGroup>
